# SKY-HOUSE

A modern responsive website layout built with HTML and CSS.

## Live Demo

https://gnonik1.github.io/SKY-HOUSE/

## Overview

SKY-HOUSE is a static frontend project focused on clean visual presentation, structured page layout, and responsive design.

The project demonstrates how a modern website interface can be built with simple core web technologies while keeping the layout organized, readable, and visually balanced.

## Features

* Modern landing page structure
* Clean and organized layout
* Responsive design approach
* Section-based page composition
* Balanced spacing and visual hierarchy
* Simple, readable HTML and CSS structure

## Technologies Used

* HTML
* CSS
* Git
* GitHub Pages

## What I Focused On

* Building a polished static website layout
* Creating clear page sections
* Working with spacing, alignment, and visual hierarchy
* Improving responsive layout structure
* Publishing the project with GitHub Pages
